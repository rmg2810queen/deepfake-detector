Deepfake Detector

Using AI to Protect Digital Identity Online

The Problem

  • AI can generate highly realistic fake human faces.
  • These images are often used for impersonation and fake profiles.
  • Deepfake identities can manipulate and mislead people online.
  • Most users cannot easily tell if a face is real or AI-generated.

My Solution

  • Trained an image classification model using Teachable Machine.
  • Detects real human faces vs AI-generated faces.
  • Deployed as a live web application using TensorFlow.js.
  • Generates probability scores and a Trust Report with safety tips.

How It Protects Digital Identity

  • Helps users detect fake profile photos before trusting accounts.
  • Reduces impersonation and scam risks.
  • Encourages critical evaluation of online identities.
  • Provides clear safety guidance based on AI probability.

How It Works

  • Trained an AI image classification model using real and AI-generated faces.
  • The model learns visual patterns and inconsistencies.
  • Runs directly in the browser — no images are stored.
  • Displays probability results and converts them into a Trust Report.

Limitations & Future Improvements

  • The model is not 100% accurate.
  • Some real images may appear AI-generated.
  • Currently only works for still images.
  • Future versions could support video deepfake detection.
